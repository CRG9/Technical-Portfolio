Work Credits: Chris Galetta

  1. Quest Creator is one of the most powerful plugins available on the Minecraft server market.
  2. This plugin provided the opportunity to create scripts and programs within the game without needing to use a single line of the Java programming language.
  3. Through this plugin, I was able to create tutorials, milestone achievements, and other task-based reward programs for players on the server to enjoy.

Below is a showcase of two introductory/interactive tutorials players were able to take when joining the server.

Video by Creator: Ribzy

"The Void" Tutorial

This in-game tutorial introduced players to the level clearing functionality of "The Void". In this tutorial, players are tasked with killing a set number of monsters before being tasked with defeating the final boss. Upon defeating the boss, the tutorial ends and the player is rewarded.
