User Interfaces

Pairing Design with Functionality

One of the most fulfilling challenges I faced on Atomix was creating user interfaces that developed a
cohesive theme for the server while still keeping functionality simple, intuitive, and effective.

The server is currently offline and I don't have access to display the hover actions for each image like I did in the Item Upgrade Menu :/

Item Upgrade Menu

The Upgrade Menu empowered players to upgrade their items with powerful abilities that made progressing through the game proceed much faster.

Void Conqueror Menu

The Void Conqueror progression path provided players with a progress bar to track their efforts of clearing each area of the Void. To clear an area, a player had to kill a set amount of monsters, then defeat the final boss.

Gear Repair Menu

The Gear Repair menu is a one-click menu where players click the anvil, then put their items into the GUI in order to repair them. Upon clicking repair, the player pays a price in "atoms" for the repair.

Group Choice Menu

Players on Atomix had the option to choose a group to fight for in their combat efforts - Cosmic or Ethereal. Choosing a group opened the players up to opportunities to unlock new gear, powerups, and rewards!
